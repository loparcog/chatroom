# chatroom
A basic chatroom to help practice websockets and node js programming.
More features for chat administrator could be added in the future.

# How to run
This repository contains all files needed to run this game, so just clone
the repository, and with node installed, while in the server directory, run
'node server.js', and it will be ported to your localhost:2000.
